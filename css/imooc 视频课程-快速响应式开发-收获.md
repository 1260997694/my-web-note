# imooc 视频课程-快速响应式开发-收获

## 项目结构

有重置的 css，公用的 css，pc 端的以及移动端的。

```
1. reset.css
2. common.css
3. desktop.css
4. mobile.css
```

## 工具使用

### px2rem

在移动端中，全部使用 rem 单位。这时可以使用 px2rem 的一些插件，如：px2rem-webpack-plugin

### 调试工具
