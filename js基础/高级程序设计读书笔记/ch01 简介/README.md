# JavaScript 简介

## 1.1 JavaScript 简史

出现的初衷是做表单验证。

## 1.2 JavaScript 实现

**JavaScript 的实现包含三个部分：**

1. 核心 （ECMAScript）
2. 文档对象模型 （DOM）
3. 浏览器对象模型 （BOM）

**ECMAScript 组成部分：**

1. 语法
2. 类型
3. 语句
4. 关键字
5. 保留字
6. 操作符
7. 对象

**DOM 组成部分：**

1. 映射文档的结构 (DOM1)
2. DOM 视图 (DOM2)
3. DOM 事件 (DOM2)
4. DOM 样式 (DOM2)
5. DOM 遍历和范围 (DOM2)
6. DOM 加载和保存 （DOM3）
7. DOM 验证 （DOM3）
8. 支持 XML 1.0 规范 （DOM3）

**BOM 常见组成部分：**

1. screen
2. location
3. navigator
4. history
